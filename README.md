## 🚀 Run Project

```bash
$ git clone https://github.com/jtiagosantos/frontend-template.git
$ cd frontend-template
```

2️⃣ Install dependencies:

```bash
$ yarn or npm i
```

3️⃣ Start project:

```bash
$ yarn dev or npm run dev
```

<hr>

## 🔗 Run tests

```bash
$ yarn test or npm run test
```
